# Admin runbook â€” weekly cadence

## Before Gameweek opens

- Create/open new Gameweek in the app.
- Load fixtures (or set eligible teams list).
- Confirm deadline (18:00 Europe/Dublin, day before first fixture).

## During open window

- Invite entrants (share invite links).
- Reconcile payments and create Entries per person.
- Monitor non-submitters (use "Who hasn't picked?" in admin).

## At deadline

- Lock picks (run "Lock picks" / deadline automation).
- Autopick for missing Entries (runs automatically with lock).
- Copy "Week Picks Summary" and share via WhatsApp.

## After fixtures

- Enter/confirm results in the app.
- Process results (run "Process results").
- Apply wipe-out rule if required (handled by app).
- Copy "Survivors / Eliminated" message and share via WhatsApp.

## Disputes

- View audit log in admin.
- Apply override with recorded reason if needed.
